# Numerical Computing for Machine Learning

This repository contains my study notes, practice code, and implementations while learning **Numerical Computing** as a prerequisite for **Machine Learning**.

The goal of this repository is not just to learn mathematics theoretically, but to understand the mathematical foundations that power Machine Learning algorithms.

---

## 📌 Objective

Build strong mathematical intuition required for Machine Learning by studying:

- Mathematics fundamentals
- Linear Algebra
- Calculus
- Probability & Statistics
- Numerical Methods
- Matrix Operations
- Optimization concepts

with a practical **ML-focused perspective**.
